Variables

Variables and maps determine the number of columns, and the media query point at when to use different column sizes. We use these to generate the grid classes showed above, and for the custom mixins listed below.


                            $breakpoints: (
                                    xs: 0,
                                    sm: 500px,
                                    md: 800px,
                                    lg: 1000px,
                                    xl: 1500px
                                );

                                $columns-map: (
                                    12:12,
                                    10:10
                                );
                                
                                $sizes-map: (
                                    xl: 3rem,
                                    l: 1.5rem,
                                    m: 1rem,
                                    s: 0.5rem,
                                    n: 0
                                );

Mixins

Mixins are used with the grid variables to generate CSS for a grid system and its columns.


                        @mixin makeGrid() {
                            .grid {
                                display: grid;
                                grid-template-columns: repeat(#{map-get($columns-map,12)}, 1fr);

                                &.gap {
                                    grid-gap: var(--gap-size);
                                    --gap-size: #{map-get($sizes-map,'m')};
                                }
                            }
                        }

                        @mixin makeColumns($breakpoint:'', $columns:12) {
                            @for $i from 1 through $columns {
                                .col-#{$breakpoint}-#{$i} {
                                    grid-column-end: span #{$i};
                                }
                                .offset-#{$breakpoint}-#{$i} {
                                    grid-column-start: $i;
                                }
                            }
                        }
